# Maiores Geradores de Energia do Brasil

Frontend para visualização dos maiores geradores de energia do Brasil.

## Tecnologias

- React 18
- TypeScript
- Vite
- Axios
- React Data Table Component

## Instalação

```bash
npm install
```

## Como usar

Desenvolvimento:
```bash
npm run dev
```

Build:
```bash
npm run build
```

A aplicação roda em `http://localhost:3000` e se conecta com a API em `http://localhost:8080`.

## Funcionalidades

- Listagem de geradores de energia
- Busca por nome, UF, tipo ou origem
- Ordenação por colunas
- Paginação
